# shortcut-hints Specification

## Purpose

主要な操作にキーボードショートカット（保存＝Ctrl/Cmd+Enter、閉じる＝Esc、タブ切替＝数字キー 1〜6）を提供し、それらをホバー／フォーカス時のカスタムツールチップで発見可能にする。既存の IME 変換確定 Enter ガードや単一行フォームの Enter 送信挙動を壊さないことを前提とする。

## Requirements

### Requirement: 保存ショートカット Ctrl/Cmd+Enter

保存ボタンを持つエディタ・フォーム（設定・ルール・目標作成・md エディタ・振り返り等）は、`Ctrl+Enter`（macOS では `Cmd+Enter`）で保存ボタン相当の保存操作を実行できる SHALL。既存の単一行 input での素の Enter 送信（`enter-submit-ime-guard`）は維持する。

#### Scenario: 保存ボタンのあるフォームで Ctrl+Enter 保存
- **WHEN** ユーザーが保存ボタンを持つフォーム内で `Ctrl+Enter`（または `Cmd+Enter`）を押す
- **THEN** 保存ボタンの click 相当が実行され、内容が保存される

#### Scenario: 保存ボタンが無効中は二重送信しない
- **WHEN** 保存処理中で保存ボタンが disabled のときに `Ctrl+Enter` を押す
- **THEN** 保存は再実行されない（二重送信しない）

#### Scenario: IME 変換確定の Enter では保存しない
- **WHEN** IME 変換中（`isComposing` / `keyCode === 229`）に Enter を押す
- **THEN** 変換確定として扱い、保存は実行しない

### Requirement: 閉じるショートカット Esc

モーダルが開いているとき、ユーザーは `Esc` キーでそのモーダルを閉じられる SHALL。

#### Scenario: モーダルを Esc で閉じる
- **WHEN** モーダルが開いている状態で `Esc` を押す
- **THEN** そのモーダルが閉じる（`closeModal` 相当）

#### Scenario: モーダルが無いときは無反応
- **WHEN** モーダルが開いていない状態で `Esc` を押す
- **THEN** タブや画面状態は変化しない

### Requirement: タブ切替ショートカット 数字キー

トップのタブ（今日・タイムライン・カンバン・振り返り・目標・設定）は、数字キー `1`〜`6` で左から順に対応するタブへ切り替えられる SHALL。ただしテキスト入力中（input / textarea / contenteditable にフォーカス、または IME 変換中）はこのショートカットを発火させない。

#### Scenario: 数字キーで対応タブへ切替
- **WHEN** テキスト入力にフォーカスしていない状態で `3` を押す
- **THEN** 左から 3 番目のタブ（カンバン）がアクティブになる

#### Scenario: 入力中はタブ切替しない
- **WHEN** input / textarea / contenteditable にフォーカスがある、または IME 変換中に数字キーを押す
- **THEN** タブは切り替わらず、入力にその文字が反映される

#### Scenario: 修飾キー併用時は発火しない
- **WHEN** `Ctrl`/`Cmd`/`Alt` を押しながら数字キーを押す
- **THEN** タブ切替ショートカットは発火しない（ブラウザ標準操作を妨げない）

### Requirement: ホバー／フォーカスでショートカットを提示するツールチップ

ショートカットを持つボタン（保存ボタン・モーダルの閉じるボタン・各タブ）は、マウスホバーおよびキーボードフォーカス時に、そのショートカットをカスタムツールチップで提示する SHALL。ブラウザ標準の `title` 属性による提示は用いない。ツールチップ内のキー表記は `kbd` チップ風に表示する。

#### Scenario: 保存ボタンにホバーでヒント表示
- **WHEN** ユーザーが保存ボタンにマウスホバーする
- **THEN** 「保存」と `Ctrl+Enter`（`kbd` チップ）を含むツールチップが表示される

#### Scenario: キーボードフォーカスでもヒント表示
- **WHEN** ショートカット付きボタンにキーボードフォーカスが移る
- **THEN** 同じツールチップが表示される

#### Scenario: 離脱で非表示
- **WHEN** ホバー／フォーカスが外れる
- **THEN** ツールチップは非表示になる
