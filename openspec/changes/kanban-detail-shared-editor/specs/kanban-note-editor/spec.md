## ADDED Requirements

### Requirement: 複数ブロックをまたぐ範囲選択とコピー／切り取り／貼り付け

システムは、カード詳細のノートエディタにおいて、複数の行（見出し・段落・リスト・チェック・引用などのブロック）を**またいだ連続した範囲選択**を、ネイティブのテキスト選択操作（ドラッグ、ダブルクリック＋ドラッグ、Shift＋クリック、Ctrl+A による全選択）で成立させる SHALL。選択操作の途中で選択が解除・リセットされたり、単一ブロック内へ選択が制限されたりしてはならない SHALL。選択した範囲は、コピー・切り取りでクリップボードへ（テキストとして）取り出せ、貼り付けで選択範囲へ差し込める SHALL。

#### Scenario: ドラッグで複数ブロックを選択してコピー

- **WHEN** ユーザーがノートの先頭付近から下方の別ブロックまでマウスでドラッグして選択する
- **THEN** 途中のブロック境界で選択が途切れず、開始点から終了点までが 1 つの連続範囲として選択される
- **AND** その状態でコピーすると、選択したテキスト全体がクリップボードに入る

#### Scenario: Ctrl+A で本文全体を選択

- **WHEN** ユーザーがノートエディタにフォーカスした状態で Ctrl+A を押す
- **THEN** ブロック単位ではなくノート本文全体が 1 回で選択される
- **AND** その状態でコピーすると本文全体がクリップボードに入る

#### Scenario: ダブルクリック＋ドラッグで語→複数ブロックへ拡張

- **WHEN** ユーザーがある語をダブルクリックし、そのまま下方の別ブロックまでドラッグする
- **THEN** 選択が単語起点から複数ブロックへ連続して拡張され、ブロック単位で頭打ちにならない

#### Scenario: 選択範囲へ貼り付け

- **WHEN** ユーザーが複数ブロックにまたがる範囲を選択した状態でクリップボードのテキストを貼り付ける
- **THEN** 選択範囲が貼り付け内容で置き換わり、ノート本文（raw Markdown）が更新される

## MODIFIED Requirements

### Requirement: 構造編集後の編集カーソル維持

システムは、カード詳細のノート（単一 contenteditable のライブ Markdown エディタ）で編集中に、Enter による改行・リスト継続・インデント変更など**行構造が作り直される編集**を行った後も、キャレット（編集位置）を対象の論理位置に保持し、そのまま入力を継続できる SHALL。構造編集に伴う内部再描画があっても、フォーカスとキャレットが失われてはならない SHALL。

#### Scenario: Enter で改行してもカーソルが消えない

- **WHEN** ユーザーがノートのある行を編集中に Enter キーを押して改行する
- **THEN** カーソルは新しい行の先頭に移り、そのままテキストを続けて入力できる
- **AND** 改行直後にカーソルが消えることはない

#### Scenario: 改行を連続しても編集が続く

- **WHEN** ユーザーが Enter を続けて複数回押す
- **THEN** そのたびに新しい行が作られ、各改行後もカーソルは編集中の位置に保持される

## REMOVED Requirements

### Requirement: エディタ外へフォーカスが移った時のみ編集終了

**Reason**: ノートエディタを単一 contenteditable の共有エディタへ移行し、「非編集行＝静的プレビュー／編集中行＝textarea」というプレビュー⇄編集のトグルモデルを廃止した。エディタ全体が常時ライブに編集可能となり、「行 textarea の差し替え」「編集セッションの開始／終了」という概念自体が存在しなくなるため、本要件は不要になる。

**Migration**: 挙動の置き換えは以下のとおり。ユーザーデータ（`notes` の raw Markdown）に変更はなく移行作業は不要。
- 旧: 行クリックで textarea 化して編集開始、エディタ外クリックでプレビューへ戻す → 新: エディタ領域は常に編集可能で、モード切替は行わない。
- 旧: 「同一エディタ内の別入力へのフォーカス移動では編集終了しない」判定 → 新: 単一 contenteditable のため判定不要。
- 保存: 編集内容は共有エディタの変更通知（onChange）経由で従来どおり `notes` へ自動保存し、詳細を閉じる／フォーカスを外す際にも確定保存する。
